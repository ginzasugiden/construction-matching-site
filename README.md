# 建設現場監督マッチングサイト - プロトタイプ

## プロジェクト概要
建設会社と現場監督をマッチングする求人プラットフォームのプロトタイプです。
企業は求人を掲載でき、求職者は条件に合った求人を検索・応募できます。

## 機能エントリーポイント（URI）

### メインページ
| ページ | URI | 説明 | パラメータ |
|--------|-----|------|-----------|
| **トップページ** | `index.html` | サービス紹介、注目の求人表示 | なし |
| **求人一覧** | `jobs.html` | 求人検索・フィルタリング | `keyword`, `location`, `employmentType`, `experience` |
| **求人詳細** | `job-detail.html` | 個別求人の詳細情報 | `id` (必須) |

### ユーザー向けページ
| ページ | URI | 説明 |
|--------|-----|------|
| **企業向け** | `for-companies.html` | 求人掲載のご案内、料金プラン、お問い合わせ |
| **求職者向け** | `for-jobseekers.html` | 会員登録のご案内、登録フォーム、成功事例 |

### パラメータ仕様
- **keyword**: 検索キーワード（職種、企業名など）
- **location**: 勤務地フィルター（東京都、大阪府、愛知県など）
- **employmentType**: 雇用形態（正社員、契約社員など）
- **experience**: 経験年数（1年以上、3年以上など）
- **id**: 求人ID（1～15の整数）

### 使用例
```
# 東京都の求人を検索
jobs.html?location=東京都

# キーワードと地域を指定
jobs.html?keyword=現場監督&location=大阪府

# 求人ID=5の詳細を表示
job-detail.html?id=5
```

## データ構造

### 求人データモデル（jobs-data.js）
```javascript
{
    id: Number,              // 求人ID（1～15）
    title: String,           // 求人タイトル
    company: String,         // 企業名
    location: String,        // 勤務地（都道府県市区町村）
    salary: String,          // 給与範囲
    employmentType: String,  // 雇用形態
    experience: String,      // 必要経験年数
    badge: String|null,      // バッジ（急募、高収入など）
    tags: Array<String>,     // タグ（資格、特徴など）
    description: String,     // 仕事内容
    isNew: Boolean,          // 新着求人フラグ
    featured: Boolean        // 注目求人フラグ
}
```

### サンプルデータ概要
- **合計15件**の多様な求人データ
- **職種**: 商業施設、マンション、道路、工場、病院、トンネル、ホテル、ダム、学校、ショッピングモール、駅ビル、高速道路、データセンターなど
- **地域**: 東京、大阪、愛知、福岡、北海道、神奈川、埼玉、千葉、京都、静岡、岐阜など
- **給与範囲**: 月給28万円～70万円
- **経験年数**: 未経験歓迎～10年以上

## 使い方

### ローカル環境での起動
1. プロジェクトファイルをダウンロード
2. `index.html` をブラウザで開く
3. ナビゲーションメニューから各ページに移動
4. 求人一覧ページで検索・フィルタリングを試す
5. 求人詳細ページで詳細情報を確認

### 本番環境へのデプロイ
**Publishタブ**を使用してウェブサイトを公開できます。

## 主な機能の使い方

### 求人検索機能
1. トップページまたは求人一覧ページの検索バーでキーワードを入力
2. 勤務地、雇用形態、経験年数などの条件を選択
3. 「検索」ボタンをクリック
4. リアルタイムで求人一覧が更新されます

### 求人詳細の確認
1. 求人カードをクリック
2. 詳細ページで仕事内容、待遇、企業情報を確認
3. 「この求人に応募する」ボタンで応募（現在はアラート表示）

### 企業向けお問い合わせ
1. 企業向けページにアクセス
2. 料金プランを確認
3. お問い合わせフォームに必要事項を入力
4. 送信ボタンをクリック（現在はコンソールログ出力）

### 求職者登録
1. 求職者向けページにアクセス
2. 登録フォームに必要事項を入力
3. 利用規約に同意
4. 無料会員登録ボタンをクリック（現在はアラート表示）

## 注意事項

### プロトタイプの制限事項
- これは**プロトタイプ**であり、実際の求人データやユーザー認証機能は含まれていません
- 応募機能、登録機能は現在アラート表示のみ（バックエンド未実装）
