## 🎯 概要

このプロジェクトは、建設業界の現場監督と企業をマッチングする求人プラットフォームのプロトタイプです。

### 主な特徴

- 🎨 **モダンなUI/UX** - プロフェッショナルで使いやすいデザイン
- 📱 **完全レスポンシブ** - PC・タブレット・スマートフォン対応
- 🔍 **高度な検索機能** - 地域・職種・経験年数などで絞り込み
- ⚡ **高速動作** - Vanilla JavaScriptで軽量実装
- 🎭 **リアルなデータ** - 15件の詳細なサンプル求人データ

### 前提条件
- Webブラウザ（Chrome、Firefox、Safari、Edge）
- Gitがインストールされていること（任意）

### インストール手順

1. **リポジトリをクローン**
   ```bash
   git clone https://github.com/あなたのユーザー名/リポジトリ名.git
   cd リポジトリ名
   ```

2. **ブラウザで開く**
   ```bash
   # Macの場合
   open index.html
   
   # Windowsの場合
   start index.html
   
   # Linuxの場合
   xdg-open index.html
   ```

3. **Live Serverで開く（VS Code使用時）**
   - VS Codeで `index.html` を開く
   - 右クリック → "Open with Live Server"

### GitHub Pagesでの公開

1. **GitHubリポジトリの Settings に移動**
2. **左メニューから "Pages" を選択**
3. **Source**: "Deploy from a branch"
4. **Branch**: "main" / "(root)" を選択して Save
5. **数分後に公開URL**: `https://username.github.io/repository-name/`

## 📖 使い方

### 基本的な操作

#### 求人を検索する
1. トップページまたは求人一覧ページにアクセス
2. 検索バーにキーワードを入力（例: "現場監督"）
3. フィルターで条件を絞り込み
   - 勤務地: 東京都、大阪府など
   - 雇用形態: 正社員、契約社員など
   - 経験年数: 1年以上、3年以上など
4. 検索ボタンをクリック

#### 求人詳細を確認する
1. 求人一覧から興味のある求人カードをクリック
2. 詳細ページで以下の情報を確認
   - 仕事内容
   - 企業情報
   - 勤務条件
   - 待遇・福利厚生
3. 「この求人に応募する」ボタンで応募（現在はアラート表示）

#### URLパラメータで直接検索
```
# 東京都の求人を検索
jobs.html?location=東京都

# キーワードと地域を組み合わせ
jobs.html?keyword=現場監督&location=大阪府

# 求人ID=5の詳細を表示
job-detail.html?id=5
```
